# 问答题

1. 什么是 iframe?

iframe 也是html的一个标签，不过不是往页面上添加一个元素那么简单，iframe 会创建包含另外一个文档的内联框架
2. 如何在当前页面中嵌入其他页面？
* 通过iframe。
* frameset和frame结合。
3. iframe 有哪些常见应用？
* 插入广告
* 实现简单的局部刷新
4. 如何使用 a标签控制 iframe 中显示的内容？
将<a>标签的target属性设置为iframe的name。
5. 为什么越来越少的人使用 iframe？

iframe广泛的被用于在网页中插入广告，
